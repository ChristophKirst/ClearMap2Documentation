DataProcessing¶

ClearMap.ParallelProcessing.DataProcessing.ArrayProcessing

Tools for parallel processing of large arrays.

ClearMap.ParallelProcessing.DataProcessing.ConvolvePointList

Convolution on a subset of points only.

ClearMap.ParallelProcessing.DataProcessing.DevolvePointList

Converts point data into a devolved or smeared out image for visulalization and analysis purposes.

ClearMap.ParallelProcessing.DataProcessing.MeasurePointList

Measurements on a subset of points in large arrays.

ClearMap.ParallelProcessing.DataProcessing.StatisticsPointList

Perform varous statistical measurements in local rregions around a list of points.

This sub-package provides functions for parallel processing of large data for functions that cannot be split into blocks or that have a more optimal way of parallelization.
